У ході виконання лабораторної роботи було реалізовано контейнеризацію системи автоматизованого складання розкладу занять кафедри на основі генетичного алгоритму засобами Docker. Налаштовано передачу змінних середовища через прапор -e, проброс порту -p 8080:5000 та організовано мережу між сервісами app і redis через docker-compose. Відповідно до непарного варіанту використано режим MODE=comfort з параметрами: розмір популяції 100, кількість ітерацій 1000, імовірність мутації 0.1. Розроблено Flask веб-сервер з ендпоінтами /run, /status, /result та /health. Працездатність сервісу перевірено за допомогою Invoke-WebRequest у PowerShell — сервер повернув коректну відповідь зі статусом 200 OK.
